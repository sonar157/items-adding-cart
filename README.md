#Cart Quantity Tracker

#Quick review:
This is a simple web page where you can increase, decrease, and reset a cart quantity using buttons.
The number updates instantly on the screen using JavaScript.

#What it does:
* Add items (+1, +2, +3)
* Remove items (won’t go below 0)
* Reset cart to 0
* Updates the value live

#What this project shows:
* How JavaScript changes HTML content
* How button clicks trigger actions
* How to prevent invalid values (like negative quantity)

#No styling (just basic UI)
#Basic DOM practice project.
